# BachelorPointJava
This Java application was built for bachelors, where the property owner 
can post advertisement, Upload 360 image of the room and other details.
The bachelors can search advertisement based on gender and location.
There are add, edit, delete option for information related to accounts,
advertisements. There is auto 360 Degree image detector and viewer which
will automatically detect 360 image and show it on separate window. 
Room based picture separation .There are built in rules to build the
password and avoid weak passwords. Both the owner and bachelors can 
send feedback to the admin. This is a project to solve the sublet 
finding issues in Bangladesh. The project will show my excellence 
in building raw logic in java programming.
